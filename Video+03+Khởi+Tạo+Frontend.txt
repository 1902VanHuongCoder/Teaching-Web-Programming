
==============================
🚀 KHỞI TẠO FRONTEND WEBSITE BÁN SÁCH
==============================

1️⃣  TẠO DỰ ÁN REACT
-----------------------------------
👉 Cách nhanh & phổ biến hiện nay:
  ```bash
  npm create vite@latest my-app -- --template react
  cd my-app
  npm install
  npm run dev
  ```
  (⚡ Sử dụng Vite giúp khởi tạo nhanh, tối ưu hiệu suất)

👉 Cách cũ (ít dùng hơn):
  ```bash
  npx create-react-app my-app
  cd my-app
  ```

2️⃣  THÊM TAILWINDCSS ĐỂ CODE GIAO DIỆN
-----------------------------------
🛠️  Cài đặt TailwindCSS:
  ```bash
  npm install -D tailwindcss postcss autoprefixer
  npx tailwindcss init -p
  ```
  (Sau khi chạy xong sẽ có file `tailwind.config.js`)

📝  Sửa file `tailwind.config.js`:
  ```js
  export default {
    content: ["./index.html","./src/**/*.{js,ts,jsx,tsx}"],
    theme: { extend: {} },
    plugins: [],
  }
  ```

🎨  Thêm vào file `index.css`:
  ```css
  @tailwind base;
  @tailwind components;
  @tailwind utilities;
  ```

✅  Thử thêm vài class Tailwind vào component để kiểm tra hoạt động!

3️⃣  CÀI ĐẶT REACT-ROUTER-DOM (điều hướng trang)
-----------------------------------
🔗  Cài đặt:
  ```bash
  npm install react-router-dom
  ```
